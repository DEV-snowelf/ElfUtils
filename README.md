# ElfUtils

A lightweight Minecraft Fabric mod providing useful utilities and tools for players.

## Features
- AutoChatManager – automates chat messages
- MathCalcManager – performs in-game calculations

## Installation

### Requirements
- Minecraft 1.20.1 (example, replace with your version)
- Fabric Loader 0.18.2

### Installation Steps
1. Make sure Fabric Loader is installed
2. Place the `.jar` file into your `mods` folder
3. Launch Minecraft with Fabric Loader
4. Enjoy ElfUtils in-game!

## Commands
- /elfautochat true/false - enable/disable autochat
- /elfcalc true/false - enable/disable autocalc
- /elfconfig reload - reload your config  minecraft/config/autochatcommand.json
